How to be in a Clan

On 4th March 2022, Bedwars release a clan update. Many players were curious and want to join a clan. They either join an existing clan or create a new one. However, creating a clan cost 399 robux($USD 5), so more people choose to join.

What is in a Clan

There are 4 pages in a Clan; General, Member, Kit Shop, Upgrade


General shows
  1. Clan Name
  2. Clan Description
  3. Announcement
  4. Number of member
  5. Leader name
  6. Your Contribution
  7. Clan Contribution

Members shows
  1. Members Name
  2. Members Role
  3. Members Daily Contribution
  4. Members total Contribution

Kit Shop shows
  1. Kits that can be brought
    2. After buying a kit, any clan member can use that kit for a week.
Upgrade shows
  1. Clan Level
    2. Allows more feature like Kit Shop
  2. Member Cap
    3. Increase Maximum Number of Member
  3. Kit Shop
    4. Unlock new Kits in Kit Shop
  4. Battle pass XP
    5. Increase XP gain for Battle Pass

Tips to join a Clan

Most clan require you to earn 1000 coins daily. It is about 2 Wins or 4 Losts.
Enter the tryout. Try your best to stay in the last 3 team, then the leader will be likely to notice you
